SYSTEM ARCHITECTURE
================================================================================

Data Flow:
  User uploads file
       
  ExcelParser detects format
       
  Parse with appropriate handler
       
  Extract: {rows, columnHeaders, fileName}
       
  AnalyticsCalculator processes
       
  Calculate: {revenue, orders, growth, product}
       
  DataStorage saves to localStorage
       
  Redirect to dashboard
       
  Dashboard loads & displays real metrics

Smart Column Detection:
  Revenue:  "revenue", "sales", "total", "amount"
  Orders:   "orders", "quantity", "count", "qty"
  Product:  "product", "name", "item"
  
  Case-insensitive matching ensures flexible file formats

Number Parsing:
  Handles: "50000", "50,000", "RM50,000", "$50,000", etc.
  Automatically extracts numeric values
  Validates and converts to decimal numbers
